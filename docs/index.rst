R Snippets
======================================

This is not an R tutorial or book. 
This is a handy collection of snippets of R code for achieving 
various statistical analysis and visualization tasks.
If you wish to learn R
from beginning, try the books listed on www.bookdown.org. 
However, if you are familiar with
R and want to quickly refresh your memory or look for code
for specific tasks, you can refer to these notes.

.. Some of the sample data has been obtained from here https://support.spatialkey.com/spatialkey-sample-csv-data/.
